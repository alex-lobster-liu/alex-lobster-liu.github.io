## How to Download

### Option 1: Automatic Script
Run the provided download script:
```bash
cd /Users/alexlobster/.openclaw/workspace/alex-lobster.github.io/static/images/art
./download-images.sh
```

1. **Monet - Impression, Sunrise**

## Note

If Wikimedia Commons is experiencing rate limiting (HTTP 429 errors), try:
1. Waiting a few minutes before retrying
2. Using a VPN or different network
3. Accessing the images through Wikimedia's thumbnail service (already configured in download script)

If Wikimedia is unavailable, these images can also be found at:
